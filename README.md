
### 变量

##### 不会更改的变量

    统一使用 const 定义

##### 常量
    
    首字母大写 + 驼峰

##### 普通变量
    小写或则驼峰

##### 私有变量
    以下划线开头 + "普通变量"

##### dom对象
    以 $ 开头表示指向一个 dom 元素

### 函数定义形式
    以箭头函数为主, 箭头前后需留出空格
    
    需要 new 关键字的函数必须使用 class
    class 中的方法不允许使用箭头函数

---
涉及到 html 内容的命名规则都以 - 分隔
### 组件名称

    以 - 符合定义
    比如 ui-image, icon-font

### 组件属性名称

    以 - 符合定义
    比如 data-id, data-name

### class 名称
    
    以 - 符合定义
